环形链表

Author: 愚茶 Date: December 31, 2019  12:44:09 Category: leetcode

题目描述

给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

解题思路

采用快慢指针的思路，如果存在环，快指针一定会跑一圈后追上慢指针，不然快指针会走到链表末尾。
